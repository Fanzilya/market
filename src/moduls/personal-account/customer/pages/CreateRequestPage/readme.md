# Админ
 1) админ создаёт компонент
 2) компонент берётся и добавляется в схему
 3) выбирается объязательный или нет
 4) выбирается нужно ли добавлять ему текст и поле
 5) ставиться поле
 6) добавляется текс


# Пользователь
 1) выбирается чек бокс
 2) вводиться значение




 1) Админ на конструкторе создаёт схему и оборудованиями
	1) Наименование оборудования
		1) Активный или не активный (это для отображения на схеме у заказчика)
	2) Фотография оборудования
	3) координаты (X Y)
	4) Размеры (W H)
	5) ед. измерения
	6) градус поворота
2) Добавить линию отображения (галочка)
	1) длина полосы
	2) градус поворота
	3) Буквенное обозначение при нуле
