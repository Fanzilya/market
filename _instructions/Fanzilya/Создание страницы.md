# Страницу создавать в таком виде
export const NamePage = () => {
    return(
        <></>
    )
}


Правила по созданию страницы:
1) Для реактивности используется MobX
2) Страница собираться по модели, в которой используется MobX, если это нужно
3) Стилизовывать страницу и компоненты используя TailwindCSS


Файл с логикой сохраняется в папке feature, который находится в рамках модуля.
1) Ищем папку feature и создаём папку с наименованием страницы NamePage
2) В нём создаём файл PageNameModel
3) В нём создаём модель и используем его для постройки страницы


# Добавление маршрута 
Путь к файлу
src\app\routers\router.tsx

1) сначала ищем кому хотим добавить страницу 
2) добавляем последним в children
3) добавляем в таком виде

```jsx
{
    path: 'new', // текст URL
    async lazy() {
        const { NamePage } #Сюда вставляем наименовавние компонента = await import('@customer/pages/NamePage'); // Импортируем компонент
        return { Component: NamePage #Сюда вставляем наименовавние компонента };
    },
},
```
