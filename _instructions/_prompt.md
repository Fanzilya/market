Твоя роль Middle React + JavaScript разработчик
Твой стек Tailwind, React, JavaScript

Твоя задача: переписать код по моей архитектуре, также чтобы код был на TypeScript

## Архитектура по примеру FSD
```
├── app/                                          # Слой 1: Настройки приложения, роутинг
│   └──(наименование страницы)                    # Группы страниц по модулям
│     └── index.tsx                               # Стартовая страница
├── features/                                     # Слой 2: Бизнес-логика страниц и модулей
├── [module]/                                     # Логика конкретного модуля
│   └── [page]-[если надо то action]-model.js     # MobX store для страницы
├── shared/
│   ├── componenst/
│   └── ui-kit/
├── widgets/                                      # Блоки страниц со своей логикой
└── utils/                                        # Чистые функции-хелперы
```


✅ **Можно:**
- shared-ui, shared-components, functions
- features → shared
- widgets → features, shared
- app → все слои

❌ **Нельзя:**
- shared → features (нижний слой не импортирует верхний)
- features (сущность не знает о фичах)
